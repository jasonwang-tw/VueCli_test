## 專案建立
### 執行終端機指令安裝VueCli

**３.0版本 (mac版本開頭需加sudo)**
```
npm install @vue/cli
```
**2.9版本 (mac版本開頭需加sudo)**
```
npm install vue/cli
```

vue 3.0 才能支援以下指令建立新專案
```
vue create <Project name>
```
[執行後的環境設定](https://ithelp.ithome.com.tw/articles/10216606)

### 運行
```
npm run serve
```
